Как получить API-ключ

1. Перейдите на [https://app.exchangerate-api.com/sign-up](https://app.exchangerate-api.com/sign-up)  
2. Зарегистрируйтесь (бесплатный аккаунт — кредитная карта не требуется).  
3. Подтвердите email.  
4. В панели управления скопируйте ваш **API Key**.  
5. В файле `currency_converter.py` замените `ВАШ_API_КЛЮЧ` на этот ключ.

**Ограничения бесплатного тарифа:** 1500 запросов в месяц (достаточно для тестирования).

## Установка и запуск

1. Убедитесь, что установлен Python 3.7 или выше.  
2. Установите необходимую библиотеку:

3. Сохраните скрипт как `currency_converter.py`.  
4. Запустите программу:

## Использование

1. Введите **положительное число** в поле «Сумма».  
2. Выберите исходную валюту (Из) и целевую валюту (В).  
3. Нажмите **«Конвертировать»**.  
4. Результат появится ниже кнопки.  
5. Все конвертации сохраняются в файл `conversion_history.json` и отображаются в таблице истории (последние 10 записей).
